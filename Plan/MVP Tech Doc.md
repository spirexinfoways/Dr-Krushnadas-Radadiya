# MVP Technical Documentation

## Project Name

Dr Krushnadas Radadiya Portfolio Website

---

# 1. MVP Definition

The Minimum Viable Product (MVP) will include a **fully functional landing page** displaying the doctor's professional profile.

The goal of MVP is to:

Showcase expertise

Display procedures

Provide contact information

Allow appointment inquiries
